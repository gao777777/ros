# TurtleBot3 视觉循线与避障仿真技术文档

根据博客《【ROS】4.1 Turtlebot3仿真Waffle循线跟踪》的步骤，我们在ROS2 Humble和Gazebo环境中成功搭建了TurtleBot3小车的视觉循线仿真，解决了之前看不到赛道和小车的问题。在此基础上加入了激光雷达避障功能，使小车能够应对赛道中的障碍物。目前已完成基础循线功能，但沿黄线行驶得并不完美且在弯道处遇到障碍物时会出现卡顿、来回扭动的现象。

## 3. 循线算法实现
### 3.1 图像处理与 HSV 颜色提取

从博客中移植的核心思想：将摄像头图像转为 HSV 色彩空间，通过阈值提取黄色引导线，计算其质心与图像中心的偏差，作为 PID 控制器的输入。
```python
hsv = cv2.cvtColor(cv_image, cv2.COLOR_BGR2HSV)
mask = cv2.inRange(hsv, lower_yellow, upper_yellow)
M = cv2.moments(mask)
if M['m00'] > 0:
    cx = int(M['m10'] / M['m00'])
    err = cx - w/2.0
```

### 3.2 视野区域限制

为减少远处干扰，只关注图像底部一小块区域：
```python
top = int(h * roi_top_ratio)   # 5/6 高度
bot = top + roi_height         # 20 像素高度
mask[0:top, :] = 0
mask[bot:h, :] = 0
```

### 3.3 PID 控制器

采用增量式 PID，输出角速度：
```python
P = self.Kp * err
self.integral += err
I = self.Ki * self.integral
D = self.Kd * (err - self.last_error)
angular_z = -(P + I + D)
```
参数初始值：
- Kp = 0.001
- Ki = 0.0
- Kd = 0.0002

### 3.4 丢线处理

当没有检测到引导线时，小车减速并原地旋转搜索：
```python
self.twist.linear.x = 0.05
self.twist.angular.z = 0.3
```

## 4. 激光雷达避障功能

为处理赛道中的障碍物，增加了激光雷达订阅和简单的绕行动作。

### 4.1 障碍物检测

取正前方 30° 范围内的最小距离，若小于阈值（0.5 m）则触发避障：
```python
front_angles = list(range(0, 15)) + list(range(345, 360))
min_front = min(msg.ranges[i] for i in front_angles if msg.ranges[i] > 0.1)
self.obstacle_detected = (min_front < self.obstacle_distance_threshold)
```

### 4.2 避障动作

进入避障模式后，按时间阶段执行预设动作：
- 阶段0：右转 1.5 秒
- 阶段1：直行 1 秒
- 阶段2：左转 1.5 秒（总时长 4 秒）

动作完成后退出避障模式，恢复循线。

## 5. 当前运行状态与问题

### 5.1 成功部分

- 小车能在 turtlebot3_autorace_2020.launch.py 赛道中沿黄色引导线行驶。
- 激光雷达能检测到障碍物并触发避障。

5.2 存在的问题
- 在弯道处遇到障碍物时，小车会抵着障碍物来回扭动，无法有效绕过。
- 观察 Follower View 窗口发现，障碍物后方仍有引导线，但小车因避障动作不当而无法继续前进。
- 避障动作的转向方向可能与实际弯道方向不匹配，导致小车卡住。

## 6. 当前代码（关键部分）

以下为当前完整的节点代码（follower_proportional.py）：
```python
#!/usr/bin/env python3
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image, LaserScan
from geometry_msgs.msg import Twist
import cv2
import cv_bridge
import numpy as np

class Follower(Node):
    def __init__(self):
        super().__init__('follower')
        self.bridge = cv_bridge.CvBridge()
        self.image_sub = self.create_subscription(Image, '/camera/image_raw', self.image_callback, 10)
        self.scan_sub = self.create_subscription(LaserScan, '/scan', self.scan_callback, 10)
        self.cmd_vel_pub = self.create_publisher(Twist, '/cmd_vel', 10)
        self.twist = Twist()

        # PID参数
        self.Kp = 0.001
        self.Ki = 0.0
        self.Kd = 0.0002
        self.last_error = 0.0
        self.integral = 0.0

        # 颜色阈值
        self.lower_yellow = np.array([20, 43, 46])
        self.upper_yellow = np.array([90, 255, 255])

        # 视野区域
        self.roi_top_ratio = 5/6
        self.roi_height = 20

        # 避障参数
        self.obstacle_detected = False
        self.obstacle_distance_threshold = 0.5
        self.avoidance_mode = False
        self.avoidance_start_time = None
        self.avoidance_duration = 4.0

        self.get_logger().info('循线+避障节点已启动')

    def scan_callback(self, msg):
        front_angles = list(range(0, 15)) + list(range(345, 360))
        min_front = 10.0
        for i in front_angles:
            if i < len(msg.ranges) and msg.ranges[i] > 0.1:
                min_front = min(min_front, msg.ranges[i])
        self.obstacle_detected = (min_front < self.obstacle_distance_threshold)

    def image_callback(self, msg):
        try:
            cv_image = self.bridge.imgmsg_to_cv2(msg, 'bgr8')
        except Exception as e:
            return

        h, w, _ = cv_image.shape

        # 避障触发
        if self.obstacle_detected and not self.avoidance_mode:
            self.avoidance_mode = True
            self.avoidance_start_time = self.get_clock().now()

        if self.avoidance_mode:
            self._do_avoidance(cv_image)
            cv2.imshow("Follower View", cv_image)
            cv2.waitKey(3)
            return

        # 循线处理
        hsv = cv2.cvtColor(cv_image, cv2.COLOR_BGR2HSV)
        mask = cv2.inRange(hsv, self.lower_yellow, self.upper_yellow)

        top = int(h * self.roi_top_ratio)
        bot = top + self.roi_height
        mask[0:top, :] = 0
        mask[bot:h, :] = 0

        M = cv2.moments(mask)
        if M['m00'] > 0:
            cx = int(M['m10'] / M['m00'])
            err = cx - w / 2.0

            P = self.Kp * err
            self.integral += err
            I = self.Ki * self.integral
            D = self.Kd * (err - self.last_error)
            self.last_error = err
            angular_z = -(P + I + D)
            angular_z = max(-1.0, min(angular_z, 1.0))

            self.twist.linear.x = 0.15
            self.twist.angular.z = angular_z
            self.cmd_vel_pub.publish(self.twist)
        else:
            self.twist.linear.x = 0.05
            self.twist.angular.z = 0.3
            self.cmd_vel_pub.publish(self.twist)
            self.integral = 0.0

        cv2.imshow("Follower View", cv_image)
        cv2.imshow("Mask", mask)
        cv2.waitKey(3)

    def _do_avoidance(self, cv_image):
        if self.avoidance_start_time is None:
            return
        elapsed = (self.get_clock().now() - self.avoidance_start_time).nanoseconds / 1e9
        if elapsed > self.avoidance_duration:
            self.avoidance_mode = False
            return

        if elapsed < 1.5:
            self.twist.linear.x = 0.1
            self.twist.angular.z = -0.5   # 右转
        elif elapsed < 2.5:
            self.twist.linear.x = 0.15
            self.twist.angular.z = 0.0    # 直行
        else:
            self.twist.linear.x = 0.1
            self.twist.angular.z = 0.3    # 左转
        self.cmd_vel_pub.publish(self.twist)

def main(args=None):
    rclpy.init(args=args)
    follower = Follower()
    rclpy.spin(follower)
    follower.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

## 7. 下一步调试方向
1.调整避障方向：根据弯道实际方向，修改避障阶段的转向顺序。当前先右转可能不适合内侧障碍物，可改为先左转。

2.动态转向选择：利用激光雷达数据判断左侧和右侧的空旷程度，选择较安全的一侧绕行。

3.PID 参数微调：观察小车在弯道处的响应，适当增大 Kp 或 Kd 改善过弯稳定性。

4.视野区域优化：根据障碍物位置，临时调整 ROI 区域，帮助小车更快重新捕获引导线。

5.增加避障后恢复逻辑：避障结束后暂时降低速度或扩大视野，提高找线成功率。

## 8. 总结

目前已实现 TurtleBot3 在 Gazebo 中的视觉循线和基础避障功能，能够应对简单赛道。当前主要问题是避障动作不够智能，导致在弯道障碍物处卡顿。后续尽力通过参数调整和逻辑优化，使小车能够更流畅地完成循线任务。
