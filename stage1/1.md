# 环境配置与仿真环境搭建
## 一、环境配置
### 1.1操作系统与ROS版本
|项目|版本|
|---- |---- |
|操作系统|Ubuntu 22.04.5 LTS|
|ROS版本|ROS 2 Humble|
|Gazebo版本|Gazebo Fortress (ros-humble-ros-gz)|
### 1.2安装命令
```bash
# 安装ROS2 Humble桌面版
sudo apt install ros-humble-desktop

# 安装Gazebo集成
sudo apt install ros-humble-ros-gz

# 安装TurtleBot3仿真包（用于测试）
sudo apt install ros-humble-turtlebot3-gazebo
sudo apt install ros-humble-turtlebot3-description

# 安装开发工具
sudo apt install python3-colcon-common-extensions
```
### 1.3问题与解决
|问题|现象|解决方法|
|----|----|----|
|无线网卡不识别|找不到WiFi图标|安装linux-firmware并更新内核(但问题仍未解决，目前使用USB共享网络)|
|Gazebo启动卡死|黑屏无响应|使用软件渲染模式 LIBGL_ALWAYS_SOFTWARE=1|
|spawn小车失败|Service unavailable|改用 ros2 run gazebo_ros gazebo 启动，自动加载ROS插件|
## 二、仿真赛道搭建
### 2.1图片转Gazebo模型
#### 2.1.1创建模型目录结构
```bash
mkdir -p ~/.gazebo/models/my_track/materials/scripts
mkdir -p ~/.gazebo/models/my_track/materials/textures
cp ~/图片/track.png ~/.gazebo/models/my_track/materials/textures/
```
#### 2.1.2 编写材质文件 my_track.material
```bash
cat > ~/.gazebo/models/my_track/materials/scripts/my_track.material << EOF
material MyTrack/Image
{
  technique
  {
    pass
    {
      texture_unit
      {
        texture track.png
      }
    }
  }
}
EOF
```
#### 2.1.3编写模型SDF文件 model.sdf
```xml
<?xml version="1.0" ?>
<sdf version="1.6">
  <model name="my_track">
    <static>true</static>
    <link name="link">
      <visual name="visual">
        <geometry>
          <plane>
            <normal>0 0 1</normal>
            <size>100 100</size>
          </plane>
        </geometry>
        <material>
          <script>
            <uri>model://my_track/materials/scripts</uri>
            <uri>model://my_track/materials/textures</uri>
            <name>MyTrack/Image</name>
          </script>
        </material>
      </visual>
    </link>
  </model>
</sdf>
```
#### 2.1.4编写模型配置文件 model.config
```xml
<?xml version="1.0"?>
<model>
  <name>my_track</name>
  <version>1.0</version>
  <sdf version="1.6">model.sdf</sdf>
  <description>赛道图片模型</description>
</model>
```
###  2.2世界文件配置
创建 ~/gazebo_models/test.world 文件：
```xml
<?xml version="1.0" ?>
<sdf version="1.6">
  <world name="test_world">
    <include>
      <uri>model://sun</uri>
    </include>
    <include>
      <uri>model://ground_plane</uri>
    </include>
    
    <!-- 图片赛道模型 -->
    <include>
      <uri>model://my_track</uri>
      <pose>0 0 0.01 0 0 0</pose>
    </include>
  </world>
</sdf>
```
### 2.3启动Gazebo
```bash
ign gazebo ~/gazebo_models/test.world
```
## 三、小车导入
### 3.1启动Gazebo并加载ROS插件
```bash
source /opt/ros/humble/setup.bash
ros2 run gazebo_ros gazebo -- -v 4 ~/gazebo_models/test.world
```
### 3.2生成TurtleBot3小车
在新终端中执行：
```bash
source /opt/ros/humble/setup.bash
export TURTLEBOT3_MODEL=waffle
ros2 run gazebo_ros spawn_entity.py -entity turtlebot3 -file $(ros2 pkg prefix turtlebot3_description)/share/turtlebot3_description/urdf/turtlebot3_waffle.urdf
```
